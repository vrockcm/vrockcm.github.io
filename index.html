PROJECTS
Metro Map Maker
The Metro Map Maker application provides the user with a set of tools to build graphical representations of city subway systems with named lines and named stops and intersecting lines and landmarks. It also provides a means for calculating the best route to take to journey from one particular station to another using a custom path finding algorithm. It also provides an export feature such that it may export a generated map and associated metro system information to a format that can be used by a corresponding web application that will be able to make use of it. View the Bitbucket Repo.

  • Penn State Cybersecurity Competition

    My team with a total of 2 members including me took 3rd Place out of 16 participants. It was a 10 hr Cyber competition on crash forensics against memory corruption attacks sponsored by Penn State University & NSA. The contest was about finding security loopholes in software. The contest organizers produced a collection of core dumps attributable to real-world memory corruption attacks and our job was to find and pinpoint where exactly these exploits were used to corrupt memory thus locate vulnerability. We also had to perform ret2libc attacks for several challenges.
